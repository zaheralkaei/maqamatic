# Jins 'Ajam

## Overview

Jins 'Ajam has two versions depending on context. It has a major quality similar to the Western major scale.

---

## 5-Note Version (Primary)

The 5-note version of Jins 'Ajam is the most common version, and is the first jins in Maqam 'Ajam and Maqam Shawq Afza.

### Scale (on C)
```
(Baggage)        Tonic                          Ghammaz   (Baggage)
    ↓              ↓                               ↓          ↓
  (A) -- (B) ---- C -------- D -------- E -------- F -------- G ---- (A)
                       M2        M2         m2         M2
                      (200¢)    (200¢)     (100¢)     (200¢)
```

### Notes

#### Basic Scale (Tonic to Ghammaz)
| Degree | Note | Interval from previous |
|--------|------|------------------------|
| 1 (Tonic) | C | - |
| 2 | D | M2 (200¢) |
| 3 | E | M2 (200¢) |
| 4 | F | m2 (100¢) |
| 5 (Ghammaz) | G | M2 (200¢) |

#### Extended Scale (with Baggage)
| Position | Note | Role |
|----------|------|------|
| Below tonic | A | Baggage |

### Interval Structure
- **Total span**: Perfect 5th (700¢)
- **Pattern**: M2 - M2 - m2 - M2
- **Character**: Major, bright

### Usage
- First jins in Maqam 'Ajam
- First jins in Maqam Shawq Afza

---

## 3-Note Version (Secondary)

The 3-note version of Jins 'Ajam occurs mostly when it is used as a secondary jins.

### Scale (on B♭)
```
Tonic          Ghammaz
  ↓               ↓
  B♭ ------- C ------- D
        M2        M2
       (200¢)    (200¢)
```

### Notes
| Degree | Note | Interval from previous |
|--------|------|------------------------|
| 1 (Tonic) | B♭ | - |
| 2 | C | M2 (200¢) |
| 3 (Ghammaz) | D | M2 (200¢) |

### Interval Structure
- **Total span**: Major 3rd (400¢)
- **Pattern**: M2 - M2
- **Character**: Major fragment

### Usage (as secondary jins)
- On the 6th scale degree of Maqam Bayati
- On the 6th scale degree of Maqam Saba
- On the 6th scale degree of Maqam Kurd
