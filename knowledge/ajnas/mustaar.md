# Jins Musta'ar

## Overview

Jins Musta'ar is a 3-note jins. It is the first jins in Maqam Musta'ar.

---

## Scale (on E𝄳)

```
        (Baggage)   Tonic          Ghammaz   (Baggage)
            ↓         ↓               ↓          ↓
  (D#) --- E𝄳 ------ F# ----------- G ------- (A) ----- (B♭)
             M2           m2
            (200¢)       (100¢)
```

Note: E𝄳 represents E half-flat

## Notes

### Basic Scale (Tonic to Ghammaz)
| Degree | Note | Interval from previous |
|--------|------|------------------------|
| 1 (Tonic) | E𝄳 (E half-flat) | - |
| 2 | F# | M2 (200¢) |
| 3 (Ghammaz) | G | m2 (100¢) |

### Extended Scale (with Baggage)
| Position | Note | Role |
|----------|------|------|
| Below tonic | D# | Baggage (leading tone) |
| 1 | E𝄳 | Tonic |
| 2 | F# | Basic scale |
| 3 | G | Ghammaz |
| Above ghammaz | A | Baggage |
| Above ghammaz | B♭ | Baggage |

## Interval Structure

- **Total span**: Minor 3rd (300¢) from tonic to ghammaz
- **Pattern**: M2 - m2 (200¢ - 100¢)
- **Character**: Unusual, borrowed quality

## Distinctive Features

- Very small jins (only 3 notes in basic scale)
- The name "Musta'ar" means "borrowed" in Arabic
- Uses the half-flat tonic (E𝄳) characteristic of Sikah-family sounds
- The F# creates an unexpected major second from the half-flat tonic

## Usage

- **Root jins** of Maqam Musta'ar
- Creates an unusual, distinctive melodic color
- The extensive baggage notes (D#, A, B♭) expand the melodic vocabulary
