# Jins Upper Rast

## Overview

Jins Upper Rast is a 4-note version of Jins Rast, with the unusual feature (shared by Jins Upper 'Ajam) that its tonic is the last degree in the scale instead of the first. This makes it an "upper" jins, typically used in the upper register of a maqam.

---

## Scale (on G to C)

```
(Baggage)  Ghammaz                    Tonic    (Baggage)
    ↓         ↓                          ↓         ↓
   (F) ------ G -------- A -------- B𝄳 ------- C ------ (D)
         M2        M2        N2         N2        M2
        (200¢)   (200¢)    (150¢)     (150¢)    (200¢)
```

Note: B𝄳 represents B half-flat. Unlike most ajnas, the tonic is at the TOP of the scale.

## Notes

### Basic Scale (Ghammaz to Tonic)
| Degree | Note | Interval from previous |
|--------|------|------------------------|
| 1 (Ghammaz) | G | - |
| 2 | A | M2 (200¢) |
| 3 | B𝄳 (B half-flat) | N2 (150¢) |
| 4 (Tonic) | C | N2 (150¢) |

### Extended Scale (with Baggage)
| Position | Note | Role |
|----------|------|------|
| Below ghammaz | F | Baggage |
| 1 | G | Ghammaz |
| 2 | A | Basic scale |
| 3 | B𝄳 | Basic scale |
| 4 | C | Tonic |
| Above tonic | D | Baggage |

## Interval Structure

- **Total span**: Perfect 4th (500¢)
- **Pattern**: M2 - N2 - N2 (200¢ - 150¢ - 150¢)
- **Character**: Rast quality, bright but with neutral intervals, resolving upward

## Distinctive Features

- **Inverted structure**: Tonic at the top, ghammaz at the bottom
- Contains the characteristic **neutral second** (B half-flat) of the Rast family
- Two consecutive neutral seconds approaching the tonic (A-B𝄳-C)
- The melody typically ascends toward the tonic for resolution

## Comparison with Jins Rast (5-note)

| Jins | Structure | Notes | Tonic Position |
|------|-----------|-------|----------------|
| Rast (5-note) | Standard | C D E𝄳 F G | First (bottom) |
| Upper Rast (4-note) | Inverted | G A B𝄳 C | Last (top) |

## Usage

- **Second jins** in Maqam Rast
- Occurs on the **5th degree of Maqam Jiharkah**
- Used as an upper jins when the melody needs to resolve upward to a higher tonic
- Preserves the Rast character with its neutral seconds while creating upward melodic resolution
