# Iqa' Saidi 4/4

**Saidi** (Ṣa'īdī) — 4/4, 8 eighths per measure.

## Overview

> Iqa‘ Sa‘idi is a lively and quintessentially Egyptian iqa‘ from the Sa‘id region (Upper Egypt). It modulates back and forth with other lively 4/4 iqa‘at like Maqsum and Baladi.
>
> — [MaqamWorld](https://www.maqamworld.com/en/iqaa/saidi.php)

---

## Pattern

**Notation:** `D T - D D - T -`  
**Base unit:** eighth | **Total divisions:** 32 (per measure)

**Legend:** `D` = Dum (bass, accent 2) | `T` = Tak (sharp, accent 1) | `k` = Ka (ghost note, no accent) | `-` = Rest

**Events (position, duration, accent):**

```
D(0,4,2) T(4,4,1) -(8,4,0) D(12,4,1) D(16,4,2) -(20,4,0) T(24,4,1) -(28,4,0)
```

## Tempo

- **Typical:** 115 BPM
- **Range:** 90–140 BPM

## Feel

Strong, martial.

## Usage

- **Genres:** saidi, folk, tahtib
- **Regions:** Upper Egypt

## MIDI

- **Dum (D):** note 36 (C2)
- **Tak (T):** note 42 (F#2)
- **Ka (k):** note 39 (D#2)
