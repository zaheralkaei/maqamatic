# Iqa' Bambi 4/4

**Bambi** (Bambī) — 4/4, 8 eighths per measure.

## Overview

> Iqa‘ Bambi (also called Bamb) is a stretched-out Egyptian 8/4 iqa‘ that beautifully accompanies improvisations and sparse vocal sections. It is customary to start playing Bambi from the 7th beat (the dum before last), thereby emphasizing the 3 consecutive dum-s that start each cycle. An alternate form of Bambi actually starts the measure with the 3 consecutive dum-s.
>
> — [MaqamWorld](https://www.maqamworld.com/en/iqaa/bambi.php)

---

## Pattern

**Notation:** `D - T k T - D T`  
**Base unit:** eighth | **Total divisions:** 32 (per measure)

**Legend:** `D` = Dum (bass, accent 2) | `T` = Tak (sharp, accent 1) | `k` = Ka (ghost note, no accent) | `-` = Rest

**Events (position, duration, accent):**

```
D(0,4,2) -(4,4,0) T(8,4,1) k(12,2,0) T(14,2,1) -(16,4,0) D(20,4,2) T(24,4,1) -(28,4,0)
```

## Tempo

- **Typical:** 115 BPM
- **Range:** 90–140 BPM

## Feel

Lilting, egyptian folk.

## Usage

- **Genres:** folk, shaabi
- **Regions:** Egypt

## MIDI

- **Dum (D):** note 36 (C2)
- **Tak (T):** note 42 (F#2)
- **Ka (k):** note 39 (D#2)
