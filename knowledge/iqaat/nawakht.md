# Iqa' Nawakht 7/8

**Nawakht** (Nawakht) — 7/8, 7 eighths per measure.

## Overview

> Iqa‘ Nawakht is a very popular iqa‘ in the Muwashah repertoire, and tends to be played at mellow speed.
>
> — [MaqamWorld](https://www.maqamworld.com/en/iqaa/nawakht.php)

---

## Pattern

**Notation:** `D T T D T D -`  
**Base unit:** eighth | **Total divisions:** 28 (per measure)

**Legend:** `D` = Dum (bass, accent 2) | `T` = Tak (sharp, accent 1) | `k` = Ka (ghost note, no accent) | `-` = Rest

**Events (position, duration, accent):**

```
D(0,4,2) T(4,4,1) T(8,4,0) D(12,4,2) T(16,4,1) D(20,4,1) -(24,4,0)
```

## Tempo

- **Typical:** 115 BPM
- **Range:** 90–140 BPM

## Feel

Asymmetric, lively.

## Usage

- **Genres:** Turkish classical, folk
- **Regions:** Turkey

## MIDI

- **Dum (D):** note 36 (C2)
- **Tak (T):** note 42 (F#2)
- **Ka (k):** note 39 (D#2)
