# Iqa' Ciftetelli 8/8

**Ciftetelli** (Çiftetelli) — 8/8, 8 eighths per measure.

## Overview

> Iqa‘ Ciftetelli (pronounced "shaftatalli" in Arabic) is a Turkish usul that became popular in Arabic music. It is rarely used for composed melodies; instead, its primary use is to support improvisations (both vocal and instrumental). Ciftetelli is usually slow paced and stretched out over two 4/4 measures, and can share the same tempo and pace as Masmudi Kabir. Alternatively, it can be performed faster and lighter, in which case it can be notated in 4/4 and becomes very similar to Wahda w-Nuss (plus or minus some ornamentation).
>
> — [MaqamWorld](https://www.maqamworld.com/en/iqaa/ciftetelli.php)

---

## Pattern

**Notation:** `D - T - T kT -`  
**Base unit:** eighth | **Total divisions:** 32 (per measure)

**Legend:** `D` = Dum (bass, accent 2) | `T` = Tak (sharp, accent 1) | `k` = Ka (ghost note, no accent) | `-` = Rest

**Events (position, duration, accent):**

```
D(0,4,2) -(4,4,0) T(8,4,1) -(12,4,0) T(16,4,1) k(20,2,0) T(22,2,1) -(24,4,0) -(28,4,0)
```

## Tempo

- **Typical:** 80 BPM
- **Range:** 60–100 BPM

## Feel

Slow, sensual, turkish.

## Usage

- **Genres:** Turkish, belly dance
- **Regions:** Turkey, Greece, Levant

## MIDI

- **Dum (D):** note 36 (C2)
- **Tak (T):** note 42 (F#2)
- **Ka (k):** note 39 (D#2)

## Project note

MaqamWorld says Ciftetelli is usually stretched over two 4/4 measures. The project stores the single-measure form (8/8) for use as a single 8/8 measure; a separate `masmoudi_kabir` iqa' covers the slow two-measure stretched form.
