# Iqa' Masmoudi Saghir 4/4

**Masmoudi Saghir** (Masmūdī Ṣaghīr) — 4/4, 8 eighths per measure.

## Overview

> Masmoudi Saghir ("saghir" meaning little in Arabic) is the alternate name for Iqa‘ Baladi. The two names refer to the same iqa‘; MaqamWorld documents it under "Baladi" because that is the more commonly used name in Egypt.
>
> — [MaqamWorld](https://www.maqamworld.com/en/iqaa/baladi.php)

---

## Pattern

**Notation:** `D D - T D - T -`  
**Base unit:** eighth | **Total divisions:** 32 (per measure)

**Legend:** `D` = Dum (bass, accent 2) | `T` = Tak (sharp, accent 1) | `k` = Ka (ghost note, no accent) | `-` = Rest

**Events (position, duration, accent):**

```
D(0,4,2) D(4,4,1) -(8,4,0) T(12,4,1) D(16,4,2) -(20,4,0) T(24,4,1) -(28,4,0)
```

## Tempo

- **Typical:** 100 BPM
- **Range:** 80–130 BPM

## Feel

Similar to baladi, lighter.

## Usage

- **Genres:** classical, belly dance
- **Regions:** Egypt

## MIDI

- **Dum (D):** note 36 (C2)
- **Tak (T):** note 42 (F#2)
- **Ka (k):** note 39 (D#2)

## Project note

Same pattern as Baladi (`D D - T D - T -`). Kept as a separate iqa' ID because MaqamWorld documents both names; the data files have identical events.
