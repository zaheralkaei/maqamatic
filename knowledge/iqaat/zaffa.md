# Iqa' Zaffa 4/4

**Zaffa** (Zaffa) — 4/4, 8 eighths per measure.

## Overview

> Iqa‘ Zaffa is named after a wedding procession involving a loud and diverse group of percussion instruments. Aside from weddings, it is a possible modulation from slowed down versions of 4/4 iqa‘at like Baladi and Maqsum.
>
> — [MaqamWorld](https://www.maqamworld.com/en/iqaa/zaffa.php)

---

## Pattern

**Notation:** `D kT kT kD kT k`  
**Base unit:** eighth | **Total divisions:** 32 (per measure)

**Legend:** `D` = Dum (bass, accent 2) | `T` = Tak (sharp, accent 1) | `k` = Ka (ghost note, no accent) | `-` = Rest

**Events (position, duration, accent):**

```
D(0,4,2) k(4,2,0) T(6,2,1) k(8,2,0) T(10,2,1) k(12,2,0) D(14,2,2) k(16,2,0) T(18,2,1) k(20,2,0) T(22,2,0) k(24,2,0) T(26,2,1) k(28,2,0) T(30,2,0)
```

## Tempo

- **Typical:** 125 BPM
- **Range:** 100–150 BPM

## Feel

Celebratory, processional.

## Usage

- **Genres:** wedding, celebration
- **Regions:** Egypt, Levant

## MIDI

- **Dum (D):** note 36 (C2)
- **Tak (T):** note 42 (F#2)
- **Ka (k):** note 39 (D#2)
