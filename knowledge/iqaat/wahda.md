# Iqa' Wahda 4/4

**Wahda** (Waḥda) — 4/4, 8 eighths per measure.

## Overview

> Iqa‘ Wahda is the principal iqa‘ used in vocal phrases in the Tarab genre of the twentieth century. "Wahda" is Arabic for one, referring to the single dum per measure, which leaves a lot of room for the singer to ornament and gives them some control over speed.
>
> — [MaqamWorld](https://www.maqamworld.com/en/iqaa/wahda.php)

---

## Pattern

**Notation:** `D - - T - - T -`  
**Base unit:** eighth | **Total divisions:** 32 (per measure)

**Legend:** `D` = Dum (bass, accent 2) | `T` = Tak (sharp, accent 1) | `k` = Ka (ghost note, no accent) | `-` = Rest

**Events (position, duration, accent):**

```
D(0,4,2) -(4,4,0) -(8,4,0) T(12,4,1) -(16,4,0) -(20,4,0) T(24,4,1) -(28,4,0)
```

## Tempo

- **Typical:** 70 BPM
- **Range:** 50–90 BPM

## Feel

Spacious, classical.

## Usage

- **Genres:** classical, taqsim
- **Regions:** Egypt, Levant

## MIDI

- **Dum (D):** note 36 (C2)
- **Tak (T):** note 42 (F#2)
- **Ka (k):** note 39 (D#2)
