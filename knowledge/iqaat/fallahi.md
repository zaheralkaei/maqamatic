# Iqa' Fallahi 2/4

**Fallahi** (Fallāḥī) — 2/4, 4 eighths per measure.

## Overview

> Iqa‘ Fallahi is a fast-paced Egyptian iqa‘ suitable for folk music. Although it might resemble Maqsum played at double speed, it has a very different feel as Fallahi flows very smoothly without a big contrast (in accents) between dum and tak beats.
>
> — [MaqamWorld](https://www.maqamworld.com/en/iqaa/fallahi.php)

---

## Pattern

**Notation:** `DkTk D T`  
**Base unit:** eighth | **Total divisions:** 16 (per measure)

**Legend:** `D` = Dum (bass, accent 2) | `T` = Tak (sharp, accent 1) | `k` = Ka (ghost note, no accent) | `-` = Rest

**Events (position, duration, accent):**

```
D(0,2,2) k(2,2,0) T(4,2,0) k(6,2,0) D(8,4,1) T(12,4,1)
```

## Tempo

- **Typical:** 130 BPM
- **Range:** 100–160 BPM

## Feel

Rustic, bouncy.

## Usage

- **Genres:** folk, dabke
- **Regions:** Egypt, Levant

## MIDI

- **Dum (D):** note 36 (C2)
- **Tak (T):** note 42 (F#2)
- **Ka (k):** note 39 (D#2)
