# Iqa' Sudasi 6/4

**Sudasi** (Sudāsī) — 6/4, 12 eighths per measure.

## Overview

> Iqa‘ Sudasi (whose name means based on the number "six") is a very popular iqa‘ to accompany folk line dances from Lebanon and Palestine, like the Dabke. It can modulate to 3 back-to-back measures of Iqa‘ Malfuf.
>
> — [MaqamWorld](https://www.maqamworld.com/en/iqaa/sudasi.php)

---

## Pattern

**Notation:** `D T k T D D T k T D D - `  
**Base unit:** eighth | **Total divisions:** 48 (per measure)

**Legend:** `D` = Dum (bass, accent 2) | `T` = Tak (sharp, accent 1) | `k` = Ka (ghost note, no accent) | `-` = Rest

**Events (position, duration, accent):**

```
D(0,4,2) T(4,4,1) k(8,4,0) T(12,4,1) D(16,4,1) D(20,4,2) T(24,4,1) k(28,4,0) T(32,4,1) D(36,4,1) D(40,4,2) -(44,4,0)
```

## Tempo

- **Typical:** 90 BPM
- **Range:** 70–110 BPM

## Feel

Complex, flowing.

## Usage

- **Genres:** classical
- **Regions:** Egypt, Levant

## MIDI

- **Dum (D):** note 36 (C2)
- **Tak (T):** note 42 (F#2)
- **Ka (k):** note 39 (D#2)
